min-resolution-width attribute is set at the body tag to 1920. If the width of the viewport is equal to or more than 1920px, the ui controls will maintain their origin dimensions as specified. e.g. Sidebar will remain 150px wide. But if the width of the viewport reduces below 1000px, then the whole ui will be scaled horizontally to fit in the given area. e.g. if the viewport width is 500, Sidebar width will look like 75px to fit in. Similarly height will be scaled as per standard-height
